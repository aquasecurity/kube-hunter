# {{ page.vid }} - {{ page.title }}

## Issue description

The kubelet is is leaking information about runnig pods via the `/runningpods` endpoint. This endpoint is exposed as part of the kubelet's debug handlers.


## Remediation

Disable `--enable-debugging-handlers` kubelet flag.
