# {{ page.vid }} - {{ page.title }}

## Issue description

The kubelet is leaking container logs via the `/containerLogs` endpoint. This endpoint is exposed as part of the kubelet's debug handlers.


## Remediation

Disable `--enable-debugging-handlers` kubelet flag.
